# Il·luminació

Documentació de la il·luminació interior i exterior de Cal Menut.

## Criteris inicials per exterior

- 230 V, evitant transformadors quan sigui possible.
- Temperatura de color recomanada: 2700 K.
- CRI alt quan sigui rellevant.
- Grau de protecció IP65/IP66.
- Materials resistents a exterior.

## Marques considerades

Cristher, Dopo, LEDS C4, Faro Barcelona, i opcions premium com iGuzzini o ERCO.
