# Principis

## El coneixement abans que les dades

Una fotografia, un plànol o una decisió ben explicada poden ser més útils que una gràfica.

## Una sola font de veritat

Cada element existeix una sola vegada i es relaciona amb la resta.

## El context és obligatori

No documentem només què hi ha. També documentem per què es va fer així.

## La interfície ha de desaparèixer

El protagonista és la masia, no l'aplicació.
