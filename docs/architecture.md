# Arquitectura

## Stack

- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Framer Motion
- Lucide Icons
- Zod
- clsx
- date-fns

## Estructura

- `src/app`: entrada de l'aplicació
- `src/design`: tokens i fonaments visuals
- `src/components`: components reutilitzables
- `src/features`: dominis funcionals
- `src/services`: integracions externes
- `src/hooks`: hooks compartits
- `src/utils`: utilitats
